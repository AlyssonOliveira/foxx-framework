Foxx-Framework is built on ArangoDB database and Foxx services.

Create apps, develop and deploy with ease.

Why?

I wanted to create a tool for building web applications easily using Foxx & ArangoDB. I also wanted to have an easy way to build, develop my app and deploy it with ease. So I did create foxx-framework !
With foxx-framework you can create new skeleton apps with everything configurated. It comes with :

  • brunch for live reload and building assets for production
  • Riotjs for building web apps and widgets.
  • UIKit for designing views.

Get Started.

Install requirements

Foxx-framework needs nodejs and npm installed. ArangoDB of course is needed.

To deploy your application, you'll need Ruby too.

Install foxx-framework

npm install foxx-framework -g

Create your first app

You need first to create an database within ArangoDB web UI.

foxxy new sample_app --database my_test_db

This action below will create the skeleton app with a simple login authentification.

cd sample_app && npm install

This will install everything you need to run the application.

Now configure nginx

To make it work properly, we will use Nginx
brew install nginx If needed.

Edit your nginx.conf file and set : 

server {
    listen       8080;
    server_name  sampleapp.dev;

    location / {
        root   /path/to/your/folder/public/;
        index  index.html index.htm;
    }

    location /_db {
            allow all;
            proxy_pass http://127.0.0.1:8529/_db;
    }
}

Notice that in my case Nginx run on port 8080.
